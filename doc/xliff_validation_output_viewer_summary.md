### ‚úÖ check_namespace_prefixes() Test Rules

- ‚úÖ **Valid** if the document contains **no element-level namespace prefixes** such as `<ns:tag>` or `</ns:tag>`

- ‚ùå **Fail** if **any element** is prefixed with a namespace (e.g., `<ns0:child>`) ‚Äî fails fast after first instance

- ‚úÖ Attributes like `xmlns` or `xmlns:ns0` are **not checked here** ‚Äî this check focuses only on the **tag prefixing**

- ‚úÖ Reports line, column, and highlights the **namespace prefix only**

- ‚ùå Fails even if the prefix is valid XML ‚Äî KLMS does not permit prefixed tags

- ‚úÖ Works for both self-closing and paired tags
